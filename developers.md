# Developer Configuration Mapping

将开发者用户名映射到其配置文件URL的核心文件。

格式：`[username][config-file-url]`

## 添加说明

1. 创建你的配置文件（YAML格式）
2. 在上方添加映射行：`[your-username][your-config-file-url]`  
3. 提交Pull Request
4. 页面将在 `your-username.mofa.ai` 可用

详细配置说明请参考 [README.md](README.md)。
